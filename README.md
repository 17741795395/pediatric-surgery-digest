# 小儿外科前沿周报 · 网页版

## 文件
- `index.html`：单文件网页（已内嵌 2026-06-14 期数据，双击即可在浏览器打开）
- `digest.json`：周报原始数据（同目录可被 fetch 读取）

## 两种使用方式

### 方式一：本地双击（最快，零配置）
直接双击 `index.html` 即可在浏览器打开。
- ✅ 无需联网、无需服务器
- ⚠️ 显示的是 HTML 文件里嵌入的当前期数据快照
- 每周更新后需要重新生成 HTML（用 `update_web.py` 一行命令完成）

### 方式二：部署上线（推荐）
把整个 `web/` 文件夹拖到 **https://app.netlify.com/drop** 即可免费上线，30 秒搞定。
- ✅ 任何设备都能访问
- ✅ 每周更新只需要把新的 `digest.json` 替换上去，HTML 会自动 fetch 加载
- ✅ 自带 HTTPS 域名

## 更新流程（每周一）
1. 后台 CodeAct 脚本自动跑完，生成 `digests/2026-06-XX.json`
2. 把新 JSON 复制到 `web/digest.json` 覆盖旧文件
3. 重新部署到 Netlify（拖一次文件夹），新期立刻上线
